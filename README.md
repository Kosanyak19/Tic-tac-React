## Build Setup:

``` bash
# Download repository:
git clone https://github.com/Kosanyak19/Tic-tac-React.git 

# Go to the app:
cd tic-tac-minimax

# Install dependencies:
npm install

# Server with hot reload at http://localhost:3030/
npm start

# Output will be at dist/ folder
npm run build
```
